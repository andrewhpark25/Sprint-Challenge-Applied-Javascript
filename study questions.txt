1. What is the DOM?

DOM stands for Document Object Model. It is an object representation of the html elements of a webpage. 

2. What is an event? 

Events are the way users interact with the page. 

Any time there is some interaction by way of a mouse, keyboard, etc., the DOM creates and propagates an event object. 

3. What is an event listener? 

Event listener is a procedure or function that waits for an event to occur and fire a callback when the event occurs. 

4. Why would we convert a NodeList into an Array? 

It makes it easier to access and retrieve items from the NodeList.

5. What is a component? 

Components are reusable pieces of code that can be used to build elements sharing functionality and styling.
